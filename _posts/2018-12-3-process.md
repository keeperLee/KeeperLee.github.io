## 多线程



 
> 1.利用多线程相关知识控制三个线程，依照先后顺序顺序输出。

``` java

public class Task {
    public static void main(String[] args) throws Exception {
        Object A =new Object();
        Object B =new Object();
        Object C =new Object();

        MyThreadTest thread1 = new MyThreadTest("AA",C,A);
        MyThreadTest thread2 = new MyThreadTest("BB",A,B);
        MyThreadTest thread3 = new MyThreadTest("CC",B,C);
        thread1.start();
        thread1.sleep(100);
        thread2.start();
        thread2.sleep(100);
        thread3.start();
        thread3.sleep(100);
    }
}
class MyThreadTest extends Thread{
    String threadName;
    
    private Object pre;
    
    private Object self;
    
    public MyThreadTest(String threadName,Object pre,Object self) {
        this.threadName = threadName;
        this.pre = pre;
        this.self = self;
    }
    
    @Override
    public void run() {
        int count = 5;
        while(count>0) {
            synchronized(pre) {
                synchronized(self) {
                    System.out.println("Java Thread" + this.threadName);
                    count--;
                    self.notify();
                }
                try {
                    pre.wait();
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        }
        System.exit(0);
    }
}
```

> 2.利用多线程，编写一个卖票的小程序
目前有三个窗口同时出售20张票，需要你在右侧代码区域编写代码实现多个窗口售票的功能，具体要求如下：

- 票数要使用同一个静态值
- 为保证不会出现卖出同一张票，使用java多线程同步锁（synchronized或lock）。
- 创建一个站台类Station，继承Thread，重写run方法，在run方法里面执行售票操作（即如果票没卖完就一直卖）！售票要使用同步锁：即有一个站台卖这张票时，其他站台要等这张票卖完！ 

``` java
package com.review;

public class Station extends Thread{
    
    static int tick = 20;      // 1) 为了保持票数的一致，票数要静态    
    Object ob = "aa";          // 2) 创建一个协助锁定的对象，本身不参与运算，没有实际作用，就是为了帮助实现锁定语句块
    
    public Station(String name) {
        super(name);           // 通过构造方法给线程名字赋值
    }
    
    public Station() {
    }
    
    @Override
    public void run() {       // 重写run方法，实现买票操作
        while(tick>0) {
            synchronized(ob) {
                if(tick>0) {
                    System.out.println("卖出了第"+ tick + "张票");
                    tick--;
                }else {
                    System.out.println("票卖完了");
                }
            }
            try {
                sleep(100);//休息一秒
            } catch (InterruptedException e) {
                e.printStackTrace();
            }  
        }
    }
    
    public static void main(String[] args) {
        //实例化站台对象，
        Station station1=new Station();
        Station station2=new Station();
        Station station3=new Station();
    
       // 让每一个站台对象各自开始工作
        station1.start();
        station2.start();
        station3.start();
    }
}
```
